## Dockside 是什么？一句话说清楚

**Dockside 是一个「程序坞旁边的文件架」**，运行在 macOS 13（Ventura）及以上系统，支持 Intel 和 M 系列芯片的 Mac。

它在你的 Dock 两侧各显示一块面板：左侧（或顶部，取决于 Dock 位置）用来暂存文件、文件夹、链接、文字、截图等任何你拖进去的东西；右侧默认显示最近截图和下载记录，悬停即可预览。两块面板都可以高度自定义，你甚至可以把其中一块改成剪贴板历史、媒体播放控制或系统性能监测面板。

完全离线运行，不收集任何数据，经过苹果公证（Apple Notarized），隐私优先。

## Dockside 核心功能详解

### 拖放暂存：Dock 旁随手放

把任何东西拖进 Dockside 的左侧面板即可：文件、文件夹、图片、视频、PDF、链接、应用图标，甚至一段文字。

默认保存的是「引用」（快捷方式），原文件不会移动；也可以选择直接复制一份存进来。用完之后把它拖出去就行，可以选择临时移除或永久删除。

### 截图与下载：右侧随时就位

这是很多人最直接感受到价值的功能。最近截图和下载文件默认显示在右侧面板，鼠标悬停直接预览，不需要打开 Finder 或截图工具。

支持配置显示数量、更改监控的文件夹路径，也可以把这个区域替换成你常用的文件夹或「最近文件」面板。

### 自定义面板：四种模式任选

除了默认的文件暂存和截图/下载，Dockside 还支持把面板切换为：

- **剪贴板历史**：访问最近复制过的内容
- **媒体控制**：直接控制 Spotify 或 Apple Music 的播放
- **系统性能监测**：实时查看 CPU、内存等信息
- **自定义小组件面板**：按自己的习惯自由组合

### 拖放分区（Drop Zones）：触发自动化

这个功能适合有工作流需求的用户。你可以设置若干「拖放区」，把文件拖到不同区域时，自动触发不同的操作——运行 Apple Shortcuts 脚本、执行 AppleScript、压缩成 ZIP、隔空投送，等等。

把 Dockside 当作一个「文件分拣机」来用，效率提升不是一点点。

### 联动第三方应用

Dockside 可以和以下工具配合工作：

- **Clop**：一键压缩图片 / 视频 / PDF 文件大小
- **PopClip**：文字选中后快捷操作
- **Dropshare**：上传文件至云端并生成分享链接

### 多显示器与多桌面支持

跟随 Dock 位置自动适配，支持多个显示器、多个 Space、全屏 App 模式和隐藏 Dock 模式。可以固定到指定显示器。
